=====================
Partner Address label
=====================

Printable Partner Address label for partners: prints 24 labels to A4-sized
sheet of paper.

Configuration
=============
Nothing special is need to be done, but the module partner_firstname is
necessary to install, which will affect the partners somewhat.

Usage
=====
Select partners and click on Print and choose Address Label Partner.

Known issues / Roadmap
======================
